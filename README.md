# AI-Powered Candidate Screening System

An automated pipeline that extracts candidate information from raw text and syncs it directly to a recruitment database.

## 🚀 Features
- **Smart Extraction:** Uses OpenAI's GPT-4o-mini with Structured Outputs to parse CV summaries.
- **Automated Workflow:** Integrates with Make.com (Integromat) via Webhooks.
- **Live Database:** Automatically updates Google Sheets with candidate details, skills, and AI-generated summaries.
- **Robustness:** Handles missing data gracefully and prevents empty submissions.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript (Fetch API)
- **Backend:** Python, Flask
- **AI:** OpenAI API
- **Automation:** Make.com
- **Storage:** Google Sheets API
